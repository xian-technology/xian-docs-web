# Architecture

`xian-docs-web` is the public documentation site for the Xian stack.

Main areas:

- `.vitepress/`: site configuration
- `api/`, `concepts/`, `node/`, `smart-contracts/`, `tools/`: public documentation sections
- `.meta/`: internal notes about this repo’s own structure and future work

Unlike the code repos, this repo uses `.meta/` for internal notes so the site does not accidentally publish them.
